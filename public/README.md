# Folder Public untuk Logo BI News

Silakan unggah file logo asli Anda ke dalam folder ini dengan nama:
`logo_bi_news.png` (atau format lain seperti `.jpg`, `.jpeg`, `.svg`).

Sistem secara otomatis dikonfigurasi untuk membaca file logo dari path `/logo_bi_news.png` dan menampilkannya di bagian footer secara elegan.

Jika Anda belum mengunggah file tersebut, sistem akan menampilkan logo bawaan sebagai cadangan sementara yang indah.
